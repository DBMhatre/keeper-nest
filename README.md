# KeeperNest

A simple and production-grade React Native Asset Management app for BuzzyBrains Software.

## About

KeeperNest helps track company assets like laptops, monitors, and keycards, managing user roles (Admin & Employee), asset catalogs, automated return reminders, and usage reporting.

## Features

- User authentication with role-based access (Admin, Employee)
- Asset catalog management and assignment
- Automated return reminders and notifications
- Reporting on late returns and asset usage statistics

## Tech Stack

- React Native (Android-focused)
- Appwrite Backend for authentication, database, and functions
- React Navigation for app navigation

## Setup & Installation

### Prerequisites

- Node.js (LTS)
- yarn or npm
- Android Studio with SDK and emulator/device setup
